# Song Guessing Game
Website Game - Song guessing game via Apple Music 

## Functions
- Random song picking
- Search by category panel
- Skip Song Option
- 

## Programming Languages
- JavaScript

## How to launch
1. Download all files to your `XAMPP` htdocs directory
2. Launch XAMPP Appache
3. Type `localhost` to your website
4. Enjoy.
